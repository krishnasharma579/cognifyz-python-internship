# 🔐 Password Strength Checker

A Python-based Password Strength Checker that evaluates the security of a password based on common password validation rules. The program provides a password strength score along with suggestions to improve weak or medium-strength passwords.

---

## 📌 Features

- Checks minimum password length (8 characters)
- Detects uppercase letters
- Detects lowercase letters
- Detects digits
- Detects special characters
- Calculates password strength score (0–5)
- Classifies passwords as:
  - Weak
  - Medium
  - Strong
- Provides suggestions to improve password security
