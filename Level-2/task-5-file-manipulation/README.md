# 📄 File Manipulation

A Python program that reads a text file and displays useful information such as the total number of lines, words, and characters. The program also handles missing and empty files gracefully.

---

## 📌 Features

- Reads content from a text file
- Counts total lines
- Counts total words
- Counts total characters
- Handles missing files using `FileNotFoundError`
- Detects and handles empty files
- Clean and structured code

---

## 🛠️ Technologies Used

- Python 3
- File Handling
- Functions
- Exception Handling (`try-except`)
- String Methods
- OS Module
