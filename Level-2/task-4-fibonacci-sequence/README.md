# 🔢 Fibonacci Sequence Generator

A Python program that generates the Fibonacci sequence based on the number of terms entered by the user. The program validates user input and displays the sequence in a clean and readable format.

---

## 📌 Features

- Generates Fibonacci sequence up to the specified number of terms
- Accepts user input for the number of terms
- Validates integer input using `try-except`
- Ensures only positive integers are accepted
- Displays the sequence in a user-friendly format
- Clean and structured code

---

## 🛠️ Technologies Used

- Python 3
- Functions
- While Loop
- For Loop
- Conditional Statements
- Exception Handling (`try-except`)
- Input Validation
