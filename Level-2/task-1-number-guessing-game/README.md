# 🎯 Number Guessing Game

A simple Python game where the computer generates a random number between 1 and 100, and the player has to guess it. The game provides hints whether the guessed number is too high or too low until the correct number is guessed.

---

## 📌 Features

- Generates a random number between 1 and 100
- Takes user input
- Validates input
- Handles invalid inputs using exception handling
- Provides hints:
  - Too High
  - Too Low
- Counts valid attempts
- Displays the total number of attempts after guessing correctly

---

## 🛠️ Technologies Used
