# 🎯 Number Guesser

A Python-based number guessing game where the user first selects a custom range. The program generates a random number within that range, and the player must guess it with the help of hints until the correct number is found.

---

## 📌 Features

- User-defined minimum and maximum range
- Random number generation within the selected range
- Input validation
- Range validation
- Exception handling using `try-except`
- Provides hints:
  - Too High
  - Too Low
- Counts valid attempts
- Displays the total number of attempts after guessing the correct number

---

## 🛠️ Technologies Used
